¿Qué es Javascript y en qué se asemeja a Java?

JavaScript es un lenguaje de programación interpretado y orientado a objetos principalmente utilizado para agregar interactividad a páginas web en el navegador del usuario. Se utiliza principalmente para agregar interactividad y dinamismo a páginas web. Es el lenguaje de programación estándar para el desarrollo web en el lado del cliente.

La sintaxis de Java y JavaScript se asemeja. Esto se debe, entre otras cosas, a que el lenguaje de scripting se inspiró en parte en su homólogo. Sin embargo, mientras que Java se basa principalmente en C, JavaScript también se inspira en parte en Python y otros lenguajes
